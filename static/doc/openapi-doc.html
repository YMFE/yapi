服务端数据导入

基本信息

Path： /api/open/import_data

Method： POST

接口描述：

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/x-www-form-urlencoded 是

Body

参数名称 参数类型 是否必须 示例 备注
type text 是 swagger 导入方式
json text 否 {"a": 1} json 数据，类型为序列化后的字符串，请勿传递 object
merge text 是 normal 数据同步方式 normal"(普通模式) , "good"(智能合并), "merge"(完全覆盖) 三种模式
token text 是
url text 否 导入数据url，如果存在该参数，将会通过 url 方式获取数据

├─ res_bodystring非必须返回数据
├─ res_body_typestring非必须返回数据类型

枚举: json,raw

├─ uidnumber非必须用户uid
├─ add_timenumber非必须
├─ up_timenumber非必须

├─ namestring必须
├─ typestring必须
├─ examplestring必须

├─ req_paramsobject []非必须

item 类型: object

├─ namestring必须
├─ examplestring必须
├─ descstring必须

├─ namestring必须

├─ req_queryobject []非必须

item 类型: object

├─ namestring必须

├─ requiredstring必须1
